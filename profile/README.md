## 📋 Overview

<a href="#popClip">PopClip</a> is a Mac productivity tool that appears as a small action bar whenever text is selected — providing instant one-click access to text operations, search, translation, application integration, and hundreds of community-built extensions without requiring menu navigation, keyboard shortcuts, or context-switching between applications.

The <a href="#popClip">automatic popup on selection</a> is PopClip's interaction model — select any text in any Mac application and a small bar of action buttons appears at the cursor position. Actions execute with a single click on the relevant button and the popup disappears, returning focus to the original application. The entire interaction takes a second or less: select, click action, done. No menus, no modifier keys, no switching applications. The <a href="#popClip">built-in text actions</a> cover the operations performed most frequently on selected text — copy, cut, paste, open URL (when the selection is a link), search in a browser, correct spelling, change case (uppercase, lowercase, title case), count words and characters, and other fundamental text manipulation operations accessible from the popup without remembering keyboard shortcuts.

The <a href="#popClip">extension ecosystem</a> is PopClip's most powerful dimension — hundreds of community-built and official extensions add actions for specific applications and services. Send selected text to Notion, add it to Reminders, translate it through DeepL or Google Translate, look it up in a dictionary, create a new task in OmniFocus, search Amazon, convert currency, run a custom script, and hundreds of other actions. Extensions install with a click and appear in the PopClip bar when text is selected. The <a href="#popClip">extension configurability</a> customizes which extensions appear in the popup — hiding unused extensions so the popup remains compact and relevant, organizing actions to match individual workflow priorities rather than showing every installed extension on every selection.

<a href="#popClip">Search engine integration</a> opens the selected text as a search query in any configured search engine — the default browser search, specialized searches in specific platforms, and any URL pattern that accepts a query parameter. <a href="#popClip">Application sending</a> passes selected text directly to other Mac applications — sending a selection to a note-taking app, a writing tool, a translation service, or any application that accepts text input without copying and switching manually.

<a href="#popClip">Keyboard shortcut triggering</a> activates PopClip from the keyboard when the automatic popup isn't triggered — ensuring PopClip's actions are accessible in contexts where automatic selection popup is disabled or inconvenient. <a href="#popClip">Regular extension additions</a> from the active PopClip extension community continuously expand the action library — new services, new applications, and new text processing capabilities added as the Mac application ecosystem evolves.

## 💬 FAQ

<details>
<summary>What is PopClip?</summary>

PopClip is a Mac productivity application that displays a small action bar whenever text is selected, providing one-click access to text operations, search, translation, application integration, and hundreds of extension-provided actions.

</details>

<details>
<summary>How does PopClip appear?</summary>

PopClip appears automatically when text is selected in any Mac application. A small bar of action buttons appears at the cursor position. It can also be triggered with a keyboard shortcut.

</details>

<details>
<summary>What built-in actions does PopClip include?</summary>

Built-in actions include copy, cut, paste, open URL, search, correct spelling, change case, word and character count, and other fundamental text operations.

</details>

<details>
<summary>How do PopClip extensions work?</summary>

Extensions add action buttons to the PopClip bar for specific applications and services. Extensions install with a click from the PopClip extension library and appear in the bar when text is selected.

</details>

<details>
<summary>Is PopClip available on Setapp?</summary>

Yes. PopClip is available through the Setapp Mac application subscription.

</details>

<details>
<summary>Can I create custom PopClip extensions?</summary>

Yes. Custom extensions are buildable in JavaScript, shell scripts, or AppleScript for personal workflow needs.

</details>

<details>
<summary>Does PopClip work in all Mac applications?</summary>

PopClip works in most Mac applications. Some applications or contexts may suppress or override the selection popup.

</details>

<details>
<summary>Does PopClip have an iPad version?</summary>

Yes. PopClip is available for iPad with a similar extension ecosystem.

</details>

<details>
<summary>Does PopClip send text to external servers?</summary>

Actions processed locally never leave the Mac. Extensions that use external services transmit text to those services. External service extensions are identified in the extension description.

</details>
